# Rental Housing Application

A React-based rental housing app where users can add, view, search, filter, and delete rental property listings.

## Features
- Add new rental properties
- Upload & display images
- Search properties
- Filter / sort listings
- Delete listings
- Data persists using LocalStorage
- Responsive UI

## Tech Stack
React • JavaScript • CSS • LocalStorage • Git

## Component Structure
- App.jsx — Main app & state management
- AddPropertyForm.jsx — Add new properties
- PropertyList.jsx — Display properties
- PropertyCard.jsx — Individual property UI
